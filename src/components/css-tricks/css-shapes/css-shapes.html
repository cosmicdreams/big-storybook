CSS shape() Function

clip-path: shape(…) — Chrome 135+. Draw precise paths with lines, arcs, and curves using CSS-native syntax.

Lines

Rectangle line to

Four explicit line to commands trace each edge.

Diagonal Cut hline / vline

hline to and vline to draw axis-aligned edges; a final line to creates the diagonal.

Arrow line to (polygon-like)

All straight lines — shape() replaces polygon() with explicit syntax.

Arcs

Top Notch arc by … of 1px

A radius of 1px on a 2× diameter arc makes a semicircular notch — flipped inward with default ccw.

Rounded Tab arc by … cw / ccw

Alternating cw and ccw quarter-circle arcs round each corner individually.

Pie Sector arc to … of 50% large cw

Trigonometry maps a 0–100 value onto arc coordinates; large cw handles angles > 180°.

Scallop Edge arc by … small cw repeating

Multiple small cw arcs along the bottom create a decorative scalloped border.

Curves (Bézier)

Wave Bottom curve to … with Xc Yc

A quadratic Bézier: one with control point pulls the bottom edge into a smooth wave.

Slanted Shoulder line to + curve to … with

Straight lines plus a single curve command round the transition from a slanted top into the vertical edge.

Organic Blob curve to × 4

Four quadratic curves — each with a pulled-out control point — produce an asymmetric, organic blob silhouette.

Combined Techniques

Speech Bubble lines + arc corners + tail

Rounded corners via arcs; the pointy tail is a simple line to triangle.

Badge / Shield hline + arc + curve

Flat top with arc-rounded upper corners and a curve-pointed bottom — classic badge silhouette.
